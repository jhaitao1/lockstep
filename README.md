# lockstep
golang练手写的帧同步服务器

技术点：
1.kcp 协议
2.proto2数据结构协议
3.帧同步的原理
4.使用go.mod项目管理
5.redis和mysql（这块主要是想练习为主，设想的是主要账号相关的东西，看时间，没时间可能不会加）

参考模型 
    1个服务 =>可以对应N个ROOM
    1个ROOM => N个进行的Game
    1个Game => N个玩家（比如我要做的二人对战:一局Game就2个人）
    很多游戏大概都是这个样，就像QQ游戏大厅 就是一个房间有很多桌子，每一桌就是一个game
